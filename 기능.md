# 💡 로그인 및 메인화면

## 카카오톡 로그인
</br>

- 카카오톡 로그인을 이용해 회원가입 및 로그인.
- 카카오톡이 기기에 설치되어 있을 경우 바로 로그인, 설치되어 있지 않을 경우 웹 로그인으로 진입.
- 최초 로그인 이후 자동 로그인 (토큰 만료 혹은 사용자가 로그아웃을 하기 전까지)

# 💡 결제
</br>

❗ <b>테스트 결제이지만 실제로 결제되므로 주의. 24시간 이내에 자동 환불됨.</b> ❗

</br>

## 쿠폰함
</br>

- 하단의 '쿠폰함' 메뉴에서 현재 사용 가능한 쿠폰 목록 확인 가능.
- 스탬프 10개 적립 시 1장의 할인 쿠폰이 자동 발급됨. (하나의 메뉴 당 하나의 스탬프 적립.)

</br>

<figure>
    <img src="https://github.com/jjddww/AwesomeCoffee/assets/50095740/86721f99-326e-4d74-9eae-8b220b119a57", width = "360", height= "800">
</figure>

</br></br>

## 쿠폰 적용
</br>

- 결제 시 사용가능한 쿠폰이 있다면 '쿠폰 선택'을 통해 적용 가능.
- 쿠폰을 적용하여 결제할 경우엔 스탬프 적립되지 않음.
- 쿠폰 적용 후 결제할 금액이 0원이라면 결제하기 버튼을 누른 후 바로 주문 완료 화면 진입.

</br>

![쿠폰 결제](https://github.com/jjddww/AwesomeCoffee/assets/50095740/2a256545-e3d8-4250-a749-93a978f842c0)

</br></br>

## 쿠폰 미적용 후 결제
</br>

- 쿠폰을 사용하지 않고 결제 시 스탬프 적립.
- 홈 화면으로 복귀 시 스탬프가 적립된 것을 볼 수 있음.

</br>
