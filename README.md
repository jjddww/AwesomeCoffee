</br>

개발기간: 2024.02.20 ~ 2024.04.09

</br>

## ⭐️ 개요
</br>

사용자가 메뉴를 탐색하고 주문, 결제할 수 있는 커피 오더 앱.

## 🔨 기술 스택 & 라이브러리

* Jetpack
  - Compose
  - ViewModel
  - LiveData
  - Room
  - Navigation

* Retrofit2 & OkHttp3
* coroutine & coroutine flow
* Coil
* Kakao SDK (로그인 구현)
* BootPay API (결제 연동)

* 아키텍처
  - MVVM (Compose - ViewModel - Model)
  - Repository 패턴

* 서버
  - 구글 GCP (Google Cloud Platform)
  - NodeJs express
  - Mysql
